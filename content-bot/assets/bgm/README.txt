# BGM Folder Structure
# =====================
# Tambahkan file BGM (.mp3 atau .wav) di folder ini dengan naming convention:
#
# Format: [mood]_[nama].mp3
#
# Contoh:
# - energetic_upbeat.mp3
# - emotional_piano.mp3
# - funny_quirky.mp3
# - dramatic_epic.mp3
# - chill_lofi.mp3
#
# Bot akan auto-select BGM berdasarkan mood yang dideteksi LLM dari konten video.
# Jika tidak ada file yang match, akan dipilih secara random.
